# 🎬 Movie Search App

A web app that lets you search for movies and see real results with posters and ratings.

## What it does
- Search for any movie by name
- See real movie results from The Movie Database (TMDB)
- Displays movie posters, ratings and descriptions

## Built with
- Python
- Flask
- HTML & CSS
- TMDB API

## How to run it
1. Install Flask: `pip3 install flask`
2. Install requests: `pip3 install requests`
3. Add your TMDB API key to app.py
4. Run: `python3 app.py`
5. Open your browser at: `http://127.0.0.1:5000`
